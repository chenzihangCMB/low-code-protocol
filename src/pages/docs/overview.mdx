## 协议内容

本协议包含以下内容：

- 设计低代码平台的底层技术规范，与平台无关，与开发语言无关。
- 设计低代码可视化开发与传统开发模式的融合。
- 设计 DevOps 流程。
- 设计软件厂商一开、客户二开的流程，并避免冲突。
